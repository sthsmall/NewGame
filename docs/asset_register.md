# 资源登记表

使用此文件追踪项目中所有外部素材和 AI 生成素材。

不要将未登记素材导入生产场景。

## 许可偏好

首选：

- CC0 / 公共领域
- MIT / BSD / Apache 代码许可
- 具有明确商业许可的付费素材

谨慎使用：

- CC-BY
- CC-BY-SA
- GPL / LGPL 代码插件

避免：

- 无许可
- 许可不明确
- 商业游戏拆包素材
- 来源不明的转载素材
