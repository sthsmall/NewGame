# 资源登记表

使用此文件来追踪项目使用的所有外部和AI生成的资源。

请勿将未追踪的资源导入到生产场景中。

## 许可偏好

首选：

- CC0 / 公共领域
- 代码使用 MIT / BSD / Apache
- 具有明确商业许可的付费资源

谨慎使用：

- CC-BY
- CC-BY-SA
- GPL / LGPL 代码插件

避免：

- 无许可
- 许可不明确
- 盗取的资源
- 来源不明的转载资源
