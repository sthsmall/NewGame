# 初始村庄场景设计

## 目的

`start_village.tscn` 是当前原型的初始地点，用于把 Phase 1 的视觉验证和 Phase 2 的探索交互合并到一个更接近实际游戏开场的空间中。

它不是最终地图，而是一个可运行的大型场景原型，应验证：

- 玩家出生点是否能自然引导到村庄中心。
- HD-2D 相机下，民居、道路、溪流、鸟居和交互物是否可读。
- 现有 Kenney 3D 素材、手工模块和像素角色能否统一在同一画面中。
- 交互系统能否在更大的场景中继续工作。

## 场景设定

暂定地点名：水镜村。

定位：日式异世界边境村落，靠近供奉边界神的小社。村庄应有“真实生活空间”和“通往异界仪式空间”的双重感觉。

当前视觉锚点：

- 村口参道
- 两侧低矮民居
- 纸灯笼
- 溪流与小桥
- 北侧鸟居和小社入口
- 石灯笼
- 稻田或草地块
- 同行少女、告示木牌、青色护符

## 资源取舍

当前已使用：

- 自制模块化民居、鸟居、纸灯笼、石灯笼预制场景。
- Kenney Retro Fantasy Kit 的地砖、楼梯、箱子、木桶、灌木和墙体模型。
- Tiny Pixel Japan Female Character Pack 的玩家与临时 NPC 精灵。

联网调研后暂未直接导入的新资源：

- Kenney Fantasy Town Kit：CC0、160 个 3D 城镇件，适合补足村庄体量，但题材偏中世纪。
- itch.io 上存在若干 low-poly village / japan 资产包，其中不少为付费或风格不确定资源，适合作为后续候选，不作为本轮阻塞项。

本轮选择先用已有资源和手工预制搭出可玩大场景，避免等待外部素材下载影响 Phase 2 验证。

## 后续替换方向

- 将手工民居逐步替换为更明确的日式 minka / kominka 模型。
- 为鸟居、纸灯笼、石灯笼添加更细致材质或模型。
- 用更自然的地形、草丛和道路边界替换大片矩形地块。
- 为溪流、小桥和神社入口添加交互与事件触发。
- 当找到合适授权资源后，补充到 `asset_register.md`，不要只停留在候选文档中。
